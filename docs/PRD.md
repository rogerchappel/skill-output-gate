# Skill Output Gate PRD

## Goal

Provide a repeatable final preflight for completed agent work before handoff, merge, or publication.

## Non-Goals

- No live messaging
- No repository mutation
- No hosted service

## MVP Requirements

- Parse Markdown and JSON run summaries
- Require a summary, artifact evidence, and verification results
- Warn on missing risk notes or next actions
- Return JSON and Markdown reports
